Etiquetas HTML

<header>

Definición: Define el encabezado de un documento o sección.

<header>
  <h1>Título</h1>
</header>

Mi Sitio Web

Buenas prácticas: Usar un header principal por página.

<ul>

Definición: Define una lista desordenada con viñetas.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • JavaScript
  • Python
  • Java

Buenas prácticas: Usar cuando el orden no importa.

<input>

Definición: Crea un campo de entrada para datos del usuario.

<input type="text" placeholder="Nombre">

Buenas prácticas: Siempre incluir atributo 'name'.

Estructura

<nav>

Definición: Define un conjunto de enlaces de navegación.

<nav>
  <a href="/">Inicio</a>
</nav>
Inicio Acerca de

Buenas prácticas: Usar para menús principales.

<footer>

Definición: Define el pie de página.

<footer>Pie</footer>
Pie de página

Buenas prácticas: Incluir información de contacto.

Texto

Definición: Define un párrafo de texto.

<p>Este es un párrafo.</p>

Este es un ejemplo de párrafo en HTML.

Buenas prácticas: No usar solo para crear espacios.

<h1>Título principal</h1>

Título principal

Subtítulo

Buenas prácticas: Usar títulos jerárquicos para estructura semántica.

<span>texto</span>

    Tablas

    <table>

    Definición: Define una tabla.

    <table>
  <tr><td>Celda</td></tr>
</table>
    Celda

    Buenas prácticas: Usar para datos tabulares.

    <tr>

    Definición: Define una fila en una tabla.

    <tr><td>Celda</td></tr>
    Fila de ejemplo

    Buenas prácticas: Usar dentro de table.

    <td>

    Definición: Define una celda de datos.

    <td>Contenido</td>
    Dato

    Buenas prácticas: Usar dentro de tr.

    Multimedia

    <audio>

    Definición: Inserta audio.

    <audio controls></audio>

    Buenas prácticas: Incluir controles.

    <img>

    Definición: Inserta una imagen.

    <img src="imagen.jpg" alt="Descripción">

    Buenas prácticas: Incluir controles.

    <iframe>

    Definición: Inserta un contenido externo dentro de la página web (como videos, mapas, etc.).

    <iframe src="url"></iframe>

    Buenas prácticas: Usar atributos como title y allowfullscreen para mejorar accesibilidad y funcionalidad.

    Enlaces

    <a>

    Definición: Define un hipervínculo.

    <a href="url">Enlace</a>
    Enlace de ejemplo - Suscribanse

    Buenas prácticas: Usar texto descriptivo.

    <link>

    Definición: Define la relación entre el documento y un recurso externo.

    <link rel="stylesheet" href="estilo.css">

    Buenas prácticas: Usar para hojas de estilo.

    Semántica

    <main>

    Definición: Contenido principal.

    <main>Contenido</main>
    Contenido Principal

    Buenas prácticas: Solo uno por página.
