# Etiquetas HTML

## Descripción
Blog educativo simple que explica 5 etiquetas HTML fundamentales con ejemplos prácticos.

## Características

**5 etiquetas HTML explicadas:**
- `<header>` - Encabezado
- `<p>` - Párrafo
- `<ul>` - Lista desordenada
- `<input>` - Campo de entrada
- `<img>` - Imagen

**Cada etiqueta incluye:**
- Definición
- Sintaxis
- Ejemplo funcional
- Buenas prácticas

1. Abre `index.html` en tu navegador
